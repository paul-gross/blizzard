identity proposal

blizzard

A snowflake that is secretly an orchestration graph: an amber hub at the center, snow-white spokes fanning out to agent nodes at the tips. Every color is lifted from the board's own token file — the mark looks native next to the mission-control UI.

The mark — three directions

hub-flakerecommended

Snowflake anatomy, fleet semantics. Snow-white spokes with branch chevrons, cyan agent nodes at the tips, amber haloed hub. Best all-rounder.

orbit

Same skeleton, no branches, plus a faint dashed fleet ring linking the agents. More instrument-panel, less literal snow. Cleanest at large sizes.

drift

Wind-driven work: task streams blowing sideways into agent nodes. Abstract and dynamic, but least "snowflake" — a wildcard.

Favicon — survives 16px

The favicon is the full hub-flake at true proportions — chevrons, cyan tips, haloed hub — with only a hair of extra stroke weight for rasterization. Sparse and airy at 48/32px, dissolving toward dots at 16px, which is the character, not a defect. The same drawing serves the small header lockup above. Rounded-square plate in --bg keeps it legible on light browser chrome too.
