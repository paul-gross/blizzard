# Desktop mockups — the original dashboard sheets

Static HTML mockup sheets for the desktop hub/runner web dashboards, copied here
from the discovery corpus (`blizzard-discovery/discovery/mockups`) so the desktop
and [mobile](../mobile/README.md) designs live side by side. Open any file
directly in a browser.

| Sheet | Surface |
|-------|---------|
| [mission-control.html](./mission-control.html) | Hub — mission control overview |
| [flow-board.html](./flow-board.html) | Hub — flow board |
| [departures.html](./departures.html) | Hub — departures |
| [runner-panel.html](./runner-panel.html) | Runner — machine panel |
| [runner-transcripts.html](./runner-transcripts.html) | Runner — transcripts |

These are the sheets the shipped web board was designed from. Where a sheet and
the running app disagree, the app is current — the sheets are a design record,
kept here for reference when extending the dashboards.
