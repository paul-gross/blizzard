# `local-panel`

The runner-only local-panel shell — the machine-local views the `runner` app adds on
top of `fleet`: the agent roster and the session-transcript panel.

It is **not published** — it is built into the `runner` app, whose output is embedded
in the one blizzard wheel.

See [`../../README.md`](../../README.md) for the workspace: the project layout, the
scripts (lint, the vitest tier, build, client codegen), the dev server, the generated
API client, and the design layer.
